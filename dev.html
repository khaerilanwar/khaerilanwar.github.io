Infografis Kabupaten Brebes

Author membuat website untuk kebutuhan Visualisasi Data Kabupaten Brebes yang nantinya akan digunakan oleh Dinas Komunikasi Informatika dan Statistik Kabupaten Brebes. Website digunakan untuk memudahkan orang-orang mengakses Rangkuman Data yang ada di Kabupaten Brebes.

Website ini dibangun hanya dengan menggunakan HTML, CSS dan JavaScript saja. Bisa dibilang website ini termasuk web statis tapi untuk data yang disajikan bisa terbilang dinamis karena Author menggunakan Realtime Database untuk menyimpan data-data yang akan divisualisasikan.

Fitur Infografis Kabupaten Brebes

Beberapa fitur yang disediakan oleh Infografis Kabupaten Brebes yaitu :

  1. 1. Visualisasi Data

    Author berusaha mungkin memberikan Visualisasi Data yang mudah dipahami oleh semua kalangan masyarakat. Author menyajikan dalam berbagai bentuk diagram dan ringkasan data.

  2. 2. Manipulasi Data Realtime

    Author juga membuat fitur untuk memanipulasi data seperti membaca data, menambahkan data, mengubah data dan menghapus data. Ini bertujuan ketika ada penginputan yang salah, Admin yang mengelola website tersebut bisa membenarkan data yang akan disajikan.

Teknologi Infografis Kabupaten Brebes

Website Infografis Kabupaten Brebes tidak terlepas dengan yang namanya teknologi yang dibutuhkan, berikut beberapa teknologi yang dibutuhkan dalam membuat website infografis :

  • BootStrap
  • ApexCharts.js
  • Firebase Hosting
  • Firebase Realtime Database
